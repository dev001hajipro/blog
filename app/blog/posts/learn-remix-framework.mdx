## Could not create a remix tutotial project.

2024/10/29 I found bug. [Address-Book Tutorial - Failed to Install Dependencies #12642](https://github.com/remix-run/react-router/issues/12642)

```sh
$ npx create-react-router@latest --template remix-run/react-router/tutorials/address-book
```
Currently, the above command will not work. So, we need to open the package.json file and
change the react-router version to 7.1.1. and then run the below command.


```sh
$ npm install
$ npm run dev
```
